HEXO搭建博客

发表于2023-03-08|更新于2023-03-08
|阅读量:

###安装工具
Node.js下载地址
git下载地址

安装hexo

npm install hexo-cli -g

检查是否安装成功

node -v
git --version
npm -v
hexo -v

###关联GitHub
1.新建仓库

2.文件管理器中右击git bash打开git，生成SSH key

ssh-keygen -t rsa -C "GitHub注册的邮箱地址"

3.搜索打开个人用户的 .ssh 文件夹,复制 id_rsa.pub 文件的内容,粘贴到 github 上 Settting 中的 SSH and GPG keys，生成密钥

4.git bash 测试是否绑定成功

ssh -T git@github.com

###本地搭建
1.初始化hexo

hexo init

2.静态生成本地的hexo页面

hexo s

3.发布到互联网
在 hexo 的根目录下 _config.yml 文件中添加如下，并保存

deploy:
    type: git
    repository: github上对应仓库的地址
    branch: main

在 hexo 根目录下打开git bash，安装工具

npm install hexo-cli -g

###常用命令
生成静态网页

hexo g

部署到网上仓库

hexo d

清楚缓存

hexo clean

###快速开始
在hexo根目录中，安装butterfly主题风格

git clone -b master https://github.com/jerryc127/hexo-theme-butterfly.git themes/butterfly

###应用主题
修改 Hexo 根目录下的 _config.yml，把主题改为butterfly

theme:butterfly

###安装插件

npm install hexo-renderer-pug hexo-renderer-stylus --save

###主题配置文件
在 hexo 的根目录创建一个文件 _config.butterfly.yml，并把主题目录的 _config.yml 内容复制到 _config.butterfly.yml ，**_config.butterfly.yml**里的配置，其优先度较高。
